===========================================================
  ساخت فایل EXE از fw.py  بدون نیاز به نصب پایتون روی سیستم
===========================================================

محتویات این بسته:
  - fw.py                      کد اصلی برنامه
  - requirements.txt           لیست کتابخانه‌های لازم (pyperclip, pynput)
  - .github/workflows/build-exe.yml   اسکریپت ساخت خودکار EXE روی ویندوز
  - build.bat                  اسکریپت ساخت دستی (فقط اگر پایتون داشته باشی)

-----------------------------------------------------------
روش پیشنهادی (بدون پایتون) — ساخت با GitHub Actions:
-----------------------------------------------------------
1) در سایت github.com یک حساب بساز (رایگان).
2) یک ریپازیتوری جدید بساز (New repository).
3) همه‌ی فایل‌های داخل این بسته را آپلود کن
   (دقت کن پوشه‌ی .github هم آپلود شود).
4) به تب "Actions" برو؛ ساخت به‌صورت خودکار شروع می‌شود.
   (یا روی "Build Windows EXE" بزن و Run workflow را کلیک کن)
5) بعد از چند دقیقه، وارد همان اجرای workflow شو و از بخش
   "Artifacts" فایل fw-exe را دانلود کن. داخلش fw.exe هست.

نکته: این روش روی سرور ویندوزی گیت‌هاب اجرا می‌شود، پس
خروجی یک EXE واقعی ویندوز است و نیازی به پایتون روی سیستم تو نیست.

-----------------------------------------------------------
روش جایگزین (اگر بعداً پایتون نصب کردی):
-----------------------------------------------------------
1) پایتون را از python.org نصب کن (هنگام نصب تیک Add to PATH را بزن).
2) فایل build.bat را دابل‌کلیک کن.
3) فایل ساخته‌شده در مسیر dist\fw.exe قرار می‌گیرد.

-----------------------------------------------------------
نکته‌ی مهم درباره‌ی خود برنامه:
-----------------------------------------------------------
این برنامه برای ویندوز نوشته شده:
  - دکمه‌ی Forward ماوس  -> کلیپ‌بورد را فقط به ارقام تمیز می‌کند
  - دکمه‌ی Back ماوس     -> Ctrl+V می‌زند (پیست)
  - برای خروج: Ctrl+C
بعضی آنتی‌ویروس‌ها ممکن است به exe های ساخته‌شده با PyInstaller
حساسیت نشان دهند؛ این موضوع طبیعی است.
